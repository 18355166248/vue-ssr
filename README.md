# vue3-ssr

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```


### 实现服务端渲染的话 执行:
````
  npm run build:win
  node server.js
````

### 实现服务端渲染热重载的话 执行:
````
  npm run dev
````
